궁금했슈(SSU)를 설계하면서 서비스의 초기 개발 과정 중에서 log-in(재학생 전용인증 시스템)을 맡았습니다.

Sendgrid 라는 무료 이메일 배포 서비스를 이용하였습니다. 
MongoDB를 통해서 입력받은 이메일이나 데이터들을 처리하였습니다. 

전체적인 완료도는 90%입니다.
저희 조에서 개발할 서비스를 연동시켜야 하기 때문에 현재로써는 임시방편으로 chatgpt.com에 접속하도록 설정했습니다.
서비스의 색과 구조를 웹에서 구현도 하였지만 추가적인 편집이나 디자인 통일을 추후 해야할 것으로 판단되어 90%로 산정했습니다.

실행
1. 가상의 환경을 열어 전반적인 login 과정을 진행합니다(conda activate ,,,)
2. uvicorn main:app --reload 명령어 실행합니다.
3. http://127.0.0.1:8000에 접속합니다.
4. 기존에 mongoDB에 연결해둔 이메일을 입력하고 수신한 인증번호를 입력하면 인증이 되는 시스템입니다.
